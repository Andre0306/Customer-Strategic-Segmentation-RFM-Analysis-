# Customer-Strategic-Segmentation-RFM-Analysis-
🛠️ Stack Tecnológico
Python (Pandas, Numpy): Procesamiento de datos y lógica de segmentación por cuartiles.

Power BI (DAX, UI Design): Creación de un dashboard ejecutivo con principios de UI/UX.

🧠 Lógica de Negocio (RFM)
El script de Python asigna puntajes del 1 al 5 en tres dimensiones:

Recencia: Días desde la última compra.

Frecuencia: Cantidad de transacciones.

Monetario: Valor total invertido por el cliente.

📈 Visualización & UX
El dashboard fue diseñado bajo principios de Diseño de Interfaces (UI) para maximizar la legibilidad:

Treemap: Distribución rápida de la cartera.

Scatter Plot: Relación entre el valor del cliente y su inactividad.

Paleta Semántica: Uso de colores para identificar estados de alerta (En Riesgo) vs éxito (Campeones).
